# ionic-babel-webpack
An ionic boilerplate project built with webpack, written in es6

##Prerequisites

    $ npm install -g cordova ionic
    $ npm install -g webpack webpack-dev-server

##Installation

    $ npm install

##Webpack development

    $ npm start
or
    $ webpack-dev-server

##Webpack build

    $ npm run build
or
    $ webpack

*Note: this will package everything in the __www/__ folder*
